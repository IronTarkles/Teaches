## Overview

We're building bus lines for the city to try and take people to where they want to be! Also try and not break the space-time continuum!

On the side of the board, there are a number of actions. One of them depicts where people want to go. Home, work, or the pub. We'll be placing out buildings of those 3 types onto the board, building bus routes between different buildings and train stations, and most importantly, trying to get passengers where they want to go.

### Setup

As part of setup we'll place some initial buildings and bus lines to make our starting board position. We'll go over the rest of the rules before we do that.

## Round Structure

The game is made up of a series of rounds, each consisting of 2 phases. Placing action disks, and carrying out actions.

### Placing action markers

Along the right side of the board are all of the actions. 

- A turn consists of you placing a single action marker, or passing.
- Actions with a single dot, like the busses action, can only be carried out by a single player each round
- Actions with several lettered spaces can have mulitple players place action markers during a round. When you place an action marker in one of these spots, the first player to go there will place on A, the 2nd on B, and so on.
- Whenever everyone passes, we move on to carrying out actions.

- We all have 20 action markers, and they do not return to us at the end of the round
- You can place a minimum of 2 action markers, but up to as many as you want.
- If you run out of action markers, the game continues with you just watching other people take their remaining turns while you get none
- One of the ways for the game to end is if only one player has remaining action markers at the end of a round.

### Taking actions

The actions are resolved one by one, from the top to the bottom of the game board beginning with the line expansion action and ending with the turn order action.

The first person to perform an action is always the one whose action marker is furthest left.

- This means for line expansion and buildings, this will be the last player that has chosen the action
- Where the Wrrooomm and passengers action, the first player that chose the action will be the first to carry it out

A player must perform all of the actions they have chosen to their fullest extent. If an action can only be partially performed, it must be completed to the maximum extent possible. Players are not allowed to skip actions or parts of actions

### Maximum number of busses

The strength of many of the actions are determined by the "maximum number of busses"

At the start of the game, all players only have one bus, so the "maximum number of busses" is 1. 

For the line expansion, passengers, and buildings actions, things work like this:

- The player in the A space performs the action a number of times equal to the current "maximum number of busses"
- The player in the B space performs the action one less time
- The player in the C space performs the action two less times, and so on
- So if the current "maximum number of busses" was 3, for line expansion A would build 3 lines, B would build 2 lines, C would build a single line, and D and further wouldn't build any lines at all

These are called diminishing actions

### The Actions

#### Line expansion

This is how we build our bus routes. The city wants our routes to overlap as little as possible

The player in space A will build a number of bus lines equal to the "maximum number of busses", with the payer in space B building one less, the player in space C building 2 less, and so on.

Note that the furthest action marker to the left, the one that lays the fewest bus lines, goes first with the player laying the most lines going last.

Rules for expanding lines:

- A bus line can only be expanded from one of its two ends
- You must expand in a direction which is empty of other bus lines
- You are not allowed to build a line that runs parallel to another players bus line. There are 2 exceptions to this:
    - If there are no empty streets at one of the ends, you can expand your bus line parallel to another players line by placing yours right next to theirs.
    - If the end of your line ends at the end of another player's line, you may build a single stretch parallel to theirs. At the next crossing you must again select an empty street if available

#### Buses

This is how you increase the "maximum number of busses" and also get the ability to service more passengers

The player who took this action adds one of their busses to their bus section here

The "maximum number of busses" is immediately increased (meaning the rest of the actions this round use the number max)

#### Passengers

A diminishing action. Players will take the appropriate number of passengers and add them to either bus stop (in any combination)

#### Buildings

A diminishing action. Players will take the appropriate number of buildings and add them to the board. Buildings can only be added to the lowest available number building site

So if there are still sites with the number 1 visible, builds must occur there. Once the ones are gone, the 2s can be built on, and so on.

#### Clock

A player going here may choose to stop time. If they do, the needle stays on the current destination and they remove a time stone from the clock

Each time stone is worth -1 point for the player that holds it at the end of the game

If the player doesn't want to mess with time (or no one went here this round), the needle advances to the next destination.

If there are passengers on buildings, move them into the intersection

Then, if there is a building of the current destination type at an intersection, a passenger moves into the building. Each building can only hold one passenger.

#### Vrrrooomm!

This is how we get passengers to their destinations. Each passenger taken to their desired destination scores 1 point.

This action can be taken multiple times by a single player

For each action disk you resolve, deliver a number of passengers to destinations along your bus line equal to YOUR current number of busses (NOT the max)

Passengers at intersections (those not already in buildings) can be delivered to any empty building that matches the clock destination as long as there is one available at an empty building along the players train line.

## End of the game

The game ends when one of these conditions is met:

- The last time stone is taken (end game immediately, do not advance to vroom)
- All but one player run out of action disks (game ends at end of current round)
- All building sites are full (game ends at end of current round)
